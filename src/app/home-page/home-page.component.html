How to register? (Kindly read this before registration)

  1. Register individual user in the User Registration section

  2. Copy the generated User code. (This code will be used for event registration)

  3. Navigate to the Event Registration page

  4. Choose the event you want to register

  5. Add team members using the user code

  6. Register all the users on the go (If a team member wants to take part in 2 or more events register it now before going to cart for availing offers)

  7. Add all the events you want to register in the cart

  8. After adding the events, register by making the payment

**Please register in all the events that you wish to before payment to avail the combo offer**
